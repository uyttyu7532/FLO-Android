# FLO
 
### 프로그래머스 과제관 안드로이드 뮤직 플레이어 앱

## 요구사항
### 스플래시 스크린
런처 이미지를 2초간 노출 후 음악 재생 화면으로 전환

### 음악 재생 화면
- 재생중인 음악 정보 
- 현재 재생 중인 부분의 가사 하이라이팅
- Seekbar
- Play/Stop 버튼

### 전체 가사 보기 화면
-  특정 가사로 이동할 수 있는 토글 버튼
- 전체 가사 화면 닫기 버튼
- Seekbar
- Play/Stop 버튼

## 사용 라이브러리
- Jetpack
- Retrofit2
- Glide
- IndicatorSeekBar
